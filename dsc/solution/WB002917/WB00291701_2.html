為何要事先編列預算？就是預估明年度會有多少的成本支出而事先籌備。
年度預算的編列對IT部門來說，是很重要的一件事！關係著是否能爭取到錢？促使業務順利推動？更關係到許多部門（業務、行銷、製造、銷售、研發等）如何推動及執行專案計畫。

資訊部門應編列預算項目有哪些?

該如何下手編制？首先先想想有多少預算需要編列。
一般來看IT有下列這些項目要編列

  • 人事成本：包括薪資、加班、津貼、福利、年終、教育訓練
  • 硬體設備：升級、維修、租借、零件更換等
  • 軟體建置：軟體採購、改版、升級、授權等
  • 通信電信：電話費、電路租用、寬頻網絡等
  • 日常耗材：印表機用品（紙、碳粉）、光碟片
  • 差旅支出：會議、分公司出差
  • 折舊攤提：辦公設備、硬體設備、軟體系統等
  • 雜項支出：書籍/訂閱費、專業機構會員費、郵資、影印費
  • 外部服務：災難修復、異地資料備援、其他服務
  • 專案費用：專案導入費用
  • 維護修繕：機房設備維護費用

在選擇資訊系統之前.....

由於大多數的企業資訊化均不是從零開始，
因此企業期望的是適宜組織作為的資訊系統，才能讓整體效益最大化。

如何選擇資訊系統

當您在決定要導入資訊系統時，是不是也遇到以下選擇困難..........
對市場眾多資訊系統，究竟該如何選擇最適合自己的ERP系統呢？
這個業態該採用哪些套裝軟體或者需要客製化專案設計？必須同時哪些考慮軟硬體設備？
在開發工具的選擇與系統架構開放性之考量為何？提供3個評估要點給企業在選購資訊系統時參考：

資訊廠商評估

專業、體質健全的資訊廠商能協助企業順利導入ERP系統，同時也能避免企業變成資訊孤兒。可以從以下兩點來評估何謂專業、體質健全的資訊廠商。

從經營狀況

  • 廠商從過去到現在的財務狀況與發展趨勢為何？
  • 廠商的產品線是否完整？能否提供完整的資訊規劃？

從資訊服務

  • 是否有合格的系統規劃師與顧問師，提供專業的資訊規劃建議？
  • 是否有完整的教育訓練課程，協助企業熟悉系統操作？
  • 是否有專責顧問師協助輔導企業上線，降低系統導入困難度？
  • 有問題時，是否有專責窗口負責並可在電話中得到立即協助？
  • 有問題時，廠商是否有全省支援人力可協助解決問題？
  • 系統是否有持續的版本更新與升級？

系統架構、系統成熟度、系統穩定性

ERP系統的技術架構有個基本信念「在整個企業組織內，讓所有使用者使用單一的 DBMS和共通的應用程式，以及統一的使用者介面」，讓導入ERP系統的企業由內而外有一致的 User Interface 與商業資訊，因此目前的ERP系統大多採用三階層的主從式技術架構 ( 3 - tiers Client / Server )，透過前端使用者介面 ( User Interface ) 與中間的應用伺服器 ( Application Server ) 加上後端的資料庫 ( Database ) 分層管理資訊的運算與存取，也可以提升ERP系統的管理便利性與資訊回應效率。

在評估ERP系統的成熟度與穩定性，可以從產品銷售期間的來看，任何資訊軟體初上市時，系統或多或少會存在一些流程功能面〈成熟度〉或效能〈穩定性〉上的瑕疵，而經過時間的歷練後，其流程功能會更貼近業態標準〈所謂標準是業經實際驗証並廣為業界採用的觀念或作法〉，系統穩定性也會較佳。因此企業所選購之ERP系統最好符合以下兩點
  • 已經在市場上銷售一段期間。
  • 擁有一定的使用家數。

系統功能評估

若要評估ERP系統是否符合企業作業需求，可以先從ERP系統的流程功能與檢核點和業態運作標準是否大致符合？再檢視企業本身需求與系統的差異性，整理出需客製修改的部分，作出需求分析報告，這其中須避免不當或過度的客製化修改而影響系統運作與成本花費。另一方面也可多加考量資訊公司所能提供的相關業種業態管理觀念的經驗與能力，可藉由雙方的經驗與能力共同改善公司管理盲點及改進作業流程。

軟體價格向來是企業選擇系統的重要考量，但由於ERP的導入影響遍及企業的各個層面，因此若單以價格作為考量主因，一旦導入不適合的ERP，無論是機會成本或後續重新評估導入新ERP的轉換成本都會造成企業嚴重的負荷。建議企業在評估ERP軟體時可以將評估準則轉化為有權數可評量的各向量評選表格，理性且客觀地評估各資訊軟體公司所提供的ERP系統。

如何導入資訊系統

由於每家資訊廠商的E資訊RP系統皆有其本身系統架構、修改、安裝等不同的複雜程度，加上企業本身需求條件的差異，因此並無一體適用的ERP導入方式，但是在企業導入ERP過程中『顧問諮詢』對於系統導入的成效是有相當關鍵性的影響，顧問必須對於特定產業的企業營運模式及資訊科技、系統方面有相當程度的認知、了解，才能協助企業在導入ERP時尋求最適合企業運作的系統流程。 然而無論是選擇資訊公司、企管顧問公司或是企業自行規劃的系統導入服務，大致可以區分為三種導入方式：

全面性導入

將企業現有的系統完全淘汰，直接改用整套 ERP系統，藉由大幅度的改變，調整組織的營運方式與人員編制，同時達到企業流程再造的目標。
倘若事先沒規劃好，其所承擔的失敗風險也會很高，甚至帶給企業嚴重的危機，其背後面臨到的失敗成本是非常高的。

漸進式導入

選擇企業的一個單位或部門，將所需的模組作一次性的導入，待其系統運作順暢後，再進行全面性的導入，可大幅降低失敗的風險，同時可累積導入時的經驗與資源，節省全面導入的龐大花費。但是企業就必須付出等待時間，待所有單位或部門都導入後，才有一套整合性ERP系統。

快速的導入

企業依據某部分的管理、作業需求做規劃，選擇導入部份模組，等將來有需要時，再進行其他功能模組的導入，如此可達到快速導入的需求。但是背後卻也帶來缺乏整體規劃的風險，未來如果需要整合其他模組，就會面臨到重新規劃的風險成本。

ERP系統導入程序

「找對的人、做對的事」可以讓ERP系統的導入達到事半功倍的效果，企業高階主管的支持與參與、專案推行小組的推行、全體員工的執行落實及ERP廠商的專業程度，這四方面良好的協調運作是企業ERP系統能順利導入的關鍵， 然而就如同企業經營追求卓越與穩健一般，資訊系統本身也必須隨著企業經營模式的轉變成長作持續性、階段性的投入與修正，才能成為適合企業經營發展的最佳解決方案。
